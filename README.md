# CSP113 - Software Engineering 2 Prelim Reviewer

This repository contains an interactive practice assessment and reviewer for Software Engineering 2 (CSP113).

## Features
- **Interactive Quizzes**: Test your knowledge on design patterns and software engineering concepts.
- **Unified Base Code**: Review comprehensive code structures showcasing Singleton, Factory, Observer, and MVC patterns.
- **Instant Feedback**: Get immediate results and detailed explanations for your answers.

## How to use
Simply open `index.html` in your web browser to start the interactive assessment.
